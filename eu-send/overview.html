EU Send Send and receive
files in a secure way

  • based on common standards and uses existing building blocks eDelivery, eTrustEx
  • compliant with legislation
  • interoperable with other systems

EU Send is a managed service hosted and operated centrally by DIGIT or use it as building block

What is included for…

For machine interaction eDelivery

  • Based on a standardised protocol (AS4)
  • Retry mechanism and duplication elimination
  • Payload agnostic
  • Ensures integrity, authenticity and confidentiality
  • Up to 2 GB per transmission
    • Pre-processing features are also available depending on your needs
    • Flexible and configurable
    • Notifications
    • Our team provides support for integration with your systems
Show

For human interaction EU Send web

  • User-friendly interface
  • Uses EU Login for authentication
  • Payload agnostic
  • Ensures integrity, authenticity and confidentiality
  • Up to 5 GB per transmission (up to 500 MB per file)*
    • All sent or received files are available in one place
    • Customisable
    • Notifications
    • As business manager you can manage your users yourself
    • * Up to 500 MB per file and 5 GB per transmission for the web solution (eTrustEx Web) and up to 2GB per transmission for machine-to-machine (eDelivery) connection

How EU Send can
help you

Exchange any format
  • Send any file format
  • Send a bundle of files at once keeping your folder structure
  • Up to 5 GB* per transmission

*Up to 500 MB per file and 5 GB per transmission for the web solution (eTrustEx Web) and up to 2GB per transmission for machine-to-machine (eDelivery) connection

Exchange securely
  • Sensitive non-classified level of confidentiality
  • Possibility to encrypt your messages
  • Using the Secure Hosting Service of the EC Data Centre

Compliant with the EU Regulation (2018/1725) on Data Protection and Commission Decision (2017/46) on security of IT systems in the EC

Exchange with evidence
You know exactly when your files are: delivered / received, read...
